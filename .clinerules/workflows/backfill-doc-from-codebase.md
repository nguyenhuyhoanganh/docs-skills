# Backfill Doc From Codebase Workflow

Use skills `doc-context-discovery`, `doc-diagrams-as-code`, and phase authoring skills as needed.

Then:

1. Discover system behavior from existing code and operational artifacts.
2. Generate baseline requirements/HLD/DLD/API docs from evidence.
3. Verify generated docs against code evidence and revise where discrepancies are found.
4. Mark assumptions explicitly where evidence is incomplete.
5. Publish as `draft` with review tasks for owning teams.
